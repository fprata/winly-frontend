Se alguma vez pesquisou concursos públicos na Europa, encontrou códigos CPV. São a espinha dorsal de como a contratação pública da UE é organizada — e acertá-los é essencial para encontrar oportunidades relevantes.

## O Que São Códigos CPV?

CPV significa **Vocabulário Comum para os Contratos Públicos** (Common Procurement Vocabulary). É um sistema de classificação padronizado usado em todos os estados-membros da UE para descrever o objeto dos contratos públicos.

Cada código CPV é um **número de 8 dígitos** seguido de um dígito de verificação:

```
72212517-6
│││││││  │
││││││└──┤ Divisão (2 dígitos)
│││││└───┤ Grupo (3 dígitos)
││││└────┤ Classe (4 dígitos)
│││└─────┤ Categoria (5 dígitos)
│└───────┤ Subcategoria (6-8 dígitos)
└────────┘ Dígito de verificação
```

## A Hierarquia CPV

O sistema é hierárquico. Códigos de nível superior são mais abrangentes:

| Nível | Exemplo | Descrição |
|-------|---------|-----------|
| Divisão | **72** | Serviços de TI |
| Grupo | **722** | Serviços relacionados com software |
| Classe | **7221** | Consultoria de sistemas e software |
| Categoria | **72212** | Serviços de programação |
| Subcategoria | **72212517** | Software de desenvolvimento TI |

## Como Encontrar os Seus Códigos CPV

### Passo 1: Comece pelo Geral

Pense no que a sua empresa vende. É um produto (fornecimento), um serviço, ou trabalhos de construção?

- **Divisões 01-44** → Fornecimentos (bens)
- **Divisão 45** → Obras de construção
- **Divisões 50-98** → Serviços

### Passo 2: Especifique

**Divisões comuns para PMEs:**
- **48** — Pacotes de software
- **50** — Reparação e manutenção
- **71** — Serviços de arquitetura, engenharia e planeamento
- **72** — TI e serviços relacionados
- **73** — Serviços de I&D
- **79** — Serviços empresariais (consultoria, jurídico, marketing)
- **80** — Educação e formação

### Passo 3: Selecione Múltiplos Códigos

A maioria das empresas corresponde a **3-10 códigos CPV**. Não se limite a um — os compradores podem classificar necessidades semelhantes sob códigos diferentes.

Por exemplo, uma agência de desenvolvimento web pode usar:
- 72212000 — Serviços de programação de software empacotado
- 72413000 — Serviços de conceção de sítios web
- 72212517 — Software de desenvolvimento de TI
- 48000000 — Pacotes de software e sistemas de informação

## Porque São Importantes os Códigos CPV

### 1. Determinam o Que Vê

Quando pesquisa concursos no TED ou em portais nacionais, os códigos CPV são o filtro principal. Códigos errados = oportunidades perdidas.

### 2. Os Compradores Usam-nos para Encontrar Fornecedores

Muitas entidades adjudicantes pesquisam bases de dados de fornecedores por código CPV. Estar registado com os códigos corretos aumenta a sua visibilidade.

### 3. Afetam os Níveis de Concorrência

Alguns códigos CPV são altamente competitivos (ex.: serviços gerais de TI), enquanto subcategorias de nicho têm menos concorrentes. Direcionar subcategorias específicas pode melhorar a sua taxa de adjudicação.

## Erros Comuns com Códigos CPV

1. **Usar apenas códigos de nível superior** — demasiado abrangente, vai obter resultados irrelevantes
2. **Ignorar códigos suplementares** — os concursos podem ter códigos CPV principais e suplementares
3. **Não atualizar códigos** — o vocabulário CPV é periodicamente revisto
4. **Perder oportunidades entre categorias** — um projeto de consultoria pode aparecer sob 79 (serviços empresariais) ou 72 (serviços de TI)

## Códigos CPV e Correspondência por IA

Plataformas modernas de contratação pública vão além da simples correspondência CPV. Analisam o perfil completo da sua empresa — serviços, experiência passada, alcance geográfico, dimensão da equipa — e fazem a correspondência com concursos usando algoritmos de pontuação com IA.

Isto significa que não está limitado a correspondências CPV exatas. A IA pode identificar concursos relevantes mesmo quando o comprador usou uma classificação diferente da esperada.

A chave é ainda definir os seus códigos CPV com precisão como ponto de partida, e depois deixar a correspondência inteligente expandir a sua cobertura.
